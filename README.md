# Application Tracker

## Installation and Run

- ```git clone https://github.com/cmr6689/application_tracker.git```
- ```git checkout main```
- ```cd application_tracker/```
- ```cd api/```
- ```npm install```
- ```npm start```
- ```cd ../client/```
- ```npm install```
- ```npm start```

Open http://localhost:3000/ in any browser.
